## Управление игрой
* 'exit' - для выхода
* 'replay' - для перезапуска игры
* 'castling0' или 'castling7' - для рокировки по соответствующей линии
* 'move 1 1 2 3' - для передвижения фигуры с позиции 1 1 на 2 3
